# Bzip
A brute force tool for password-protected zip file/folder(s).
Note that this tool can only crack .zip files.
Please DO not misuse.
# Installation
```
git clone https://github.com/4anonz/Bzip.git
cd Bzip
pip3 install -r requirements.txt
python3 bzip.py
```
